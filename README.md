# TP Enigma

Le code pour le TP Enigma avec du multiprocessing.

# Installation

1. Clone the repository
2. Run `pip install -r requirements.txt`
3. Run `py main.py`
4. Enjoy

# Performance

Proc : I5 9600K 3.7Ghz
